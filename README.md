nbd-3dge
========

No Big Deal 3D Game Engine 

This project is created to have fun while experimenting with modern OpenGL
and C++.

Requirements
------------

* C++14
* OpengGL 3.3
* CMake 3.2

Dependencies
------------

* SDL2
* SDL2_image
* Boost
* GLM (OpenGL Mathematics)
* Bullet
